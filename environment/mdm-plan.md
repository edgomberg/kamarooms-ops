# MDM Plan — Mosyle Fuse

## Почему Mosyle Fuse / Why Mosyle Fuse

- Бесплатно до 30 устройств (нам нужно 5-7)
- Полноценный MDM для macOS
- Удалённая установка приложений
- Управление Wi-Fi профилями
- Удалённая блокировка/стирание при потере

## Процесс регистрации устройства / Enrollment Flow

1. Администратор создаёт аккаунт на business.mosyle.com
2. Скачивает Mosyle Fuse Agent на новый Mac
3. Запускает агент, вводит код организации
4. Устройство появляется в панели управления
5. MDM профиль устанавливается автоматически

## Профиль приложений / App Deployment Profile

| Приложение | Тип установки | Обязательное |
|-----------|---------------|-------------|
| Google Chrome | MDM push | Да |
| Keeper | MDM push | Да |
| Telegram Desktop | MDM push | Да |
| Yandex Browser | MDM push | Нет |

## Сетевые требования / Network Requirements

- Устройства должны иметь доступ к *.mosyle.com (порт 443)
- Apple Push Notification Service (APNs): порты 2195, 2196, 5223
- Mikrotik: создать отдельное правило для MDM-трафика

## Безопасность / Security

- FileVault шифрование включено через MDM
- Автообновление macOS включено
- Блокировка экрана через 5 минут
- Удалённое стирание доступно администратору
