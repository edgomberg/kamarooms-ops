# Стандартная конфигурация Mac / Standard Mac Setup

## Модели по ролям / Models by Role

### Стационарные (офис) / Desktop (office)
- **Mac mini M4** — 16 GB RAM, 256 GB SSD
- Цена: 54,000-57,000 ₽ (серый рынок), 72,000-75,000 ₽ (официальный)
- Для: бухгалтерия, ресепшн, офис-менеджер, администрация
- Нужен внешний монитор, клавиатура, мышь

### Мобильные (руководство) / Laptop (management)
- **MacBook Air M4** — 16 GB RAM, 256 GB SSD
- Цена: 76,000-91,000 ₽
- Для: генеральный менеджер, отдел продаж, маркетинг

## Базовый набор приложений / Base App List

1. **Chrome** — основной браузер, 1С веб-клиент
2. **1С веб-клиент** — закладка в Chrome (после миграции с тонкого клиента)
3. **Keeper** — менеджер паролей (уже развёрнут)
4. **Telegram Desktop** — рабочая коммуникация
5. **Yandex Browser** — для сервисов Яндекса (Метрика, 360)

## Настройка / Setup

1. Распаковка + первый вход с Apple ID организации
2. Регистрация в Mosyle Fuse (MDM)
3. Установка Chrome, Keeper, Telegram Desktop через MDM
4. Создание закладки 1С веб-клиент
5. Настройка Wi-Fi профиля (рабочая сеть)
6. Проверка скорости интернета на рабочем месте
